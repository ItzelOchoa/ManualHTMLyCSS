Manual de HTML y CSS
    HTML CSS

La programación es una disciplina muy amplia que ofrece cientos de posibilidades a la hora de configurar una web.

Uno de sus componentes más conocidos es HTML, pues es el lenguaje de marcado a partir del cual se define todo el contenido que vemos en las páginas, y junto a él está el CSS, orientado hacia su diseño y apariencia. De hecho, aunque muchas personas desconocen qué es HTML y CSS, es muy probable que disfruten a diario de sus aplicaciones al navegar por la red.
